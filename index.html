Baywheel Ride Cost and
Revenue for 2020 to 2021

This project involves using Python and SQL Alchemy to upload Baywheel ride data, which is available publicly on their site by year, to a PostgreSQL database. Once the data is uploaded, two SQL scripts are created. The first script calculates the cost per ride session, providing valuable insights into the pricing structure and overall cost-effectiveness of the Baywheel service. The second script aggregates all the ride session data by year and month, showing how much ride time, distance traveled, and cost for each date. This information can be used to analyze trends over time, identify peak usage periods, and optimize operations to maximize efficiency and profitability. Overall, this project demonstrates the power of data analysis and visualization in optimizing business operations and improving customer experiences.
